WHAT HAPPENS ON RAILWAY
════════════════════════════════════════════════════════════════

1. Railway reads Procfile → knows to run "python app.py"
2. Railway reads nixpacks.toml → installs Cairo/Pango/etc.
3. Railway reads requirements.txt → installs Python packages
4. Railway sets PORT environment variable → app listens on it
5. Railway provides HTTPS URL → app is accessible
6. If volume mounted at /data → app uses it for storage
